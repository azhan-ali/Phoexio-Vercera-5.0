## ✨ The Story Behind Phoenix

> *"Every human has three fires inside them — the way they **look**, the way they **fuel themselves**, and the way they **feel**. When one flickers, the others dim. Phoenix is the first AI that tends to all three."*

Most lifestyle apps treat you like a machine — cold dashboards, generic advice, endless notifications. **Phoenix is different.** It feels like flipping through the diary of a kind friend who happens to be powered by state-of-the-art AI. Wobbly borders, handwritten fonts, doodles in the margins, tape at the corners — every pixel is designed to remind you that **becoming a better version of yourself is a creative act, not a chore.**

| 🎨 Fashion Suite | 🍲 Food Suite | 🧠 Mental Health Suite |

| Stop staring at your closet | Stop wasting groceries | Stop suffering in silence |

It replaces **three separate ₹800/month apps** — a stylist, a nutritionist, and a wellness coach — with one beautifully crafted, AI-first experience. **All priced in ₹ for Indian users.**

## 🎨 Why Phoenix Feels Different

- **🖋️ Hand-drawn UI** — Every border wobbles (custom SVG filters), every card has tape corners, fonts are handwritten (`Caveat`, `Kalam`, `Patrick Hand`). It feels like a notebook, not a dashboard.
- **🎬 Cinematic Motion** — Framer Motion everywhere: parallax doodles that follow your cursor, scroll-triggered reveals, animated counters, a custom cursor with follower.
- **🔥 Giant Animated Phoenix Hero** — A living logo with pulsing flame aura, floating tags, and draw-in underlines. Source JPG is processed into a transparent PNG by `scripts/process-logo.js` (using `sharp`) at build time.
- **🧭 Scroll Progress + Wavy Dividers** — Every section separated by hand-drawn SVG paths that draw themselves in as you scroll.
- **🎭 Persona-Aware AI Chat** — The companion's whole personality switches based on persona + tone sliders. A `Friend + Funny` response reads completely differently from a `Coach + Tough Love` one — same page, same private space.
- **📱 Fully Responsive** — Floating glass navbar, slide-out drawer nav, touch-friendly tiles, mobile-optimized chat and journal.
- **♿ Accessible by Default** — Semantic HTML, keyboard nav, reduced-motion respected, color contrast AA+.

## ⚡ Quick Start (3 minutes)

### Prerequisites
- **Node.js 20+** and **npm**
- **xAI API key** → grab one at [x.ai](https://x.ai)

### 1. Clone & Install
```bash
git clone https://github.com/azhan-ali/Phoexio-Vercera-5.0.git phoenix
cd phoenix
npm install
```

### 2. Configure Environment
```bash
cp .env.example .env.local
```

Open `.env.local` and paste your key:
```ini
GROK_API_KEY=xai-xxxxxxxxxxxxxxxxxxxxxxxx
GROK_API_BASE_URL=https://api.x.ai/v1

# Current xAI models (grok-2-* names are deprecated)
GROK_TEXT_MODEL=grok-4-1-fast-non-reasoning
GROK_VISION_MODEL=grok-4-1-fast-non-reasoning
GROK_IMAGE_MODEL=grok-imagine-image
```

### 3. (Optional) Re-process the logo
If you ever replace `public/phoenix-logo.jpg`:
```bash
node scripts/process-logo.js
```
Generates a white-background-removed, tight-cropped `phoenix-logo.png`.

### 4. Run
```bash
npm run dev
```
Open **http://localhost:3000** and watch the phoenix rise. 🔥

### 5. Production Build
```bash
npm run build && npm start
```

## 🗺️ API Reference

All routes are `POST` and return strict JSON. Errors return `400/500` with `{ error: string }`.

| Endpoint | Purpose |
|---|---|
| `POST /api/fashion/outfit-analyzer` | Analyze uploaded outfit image |
| `POST /api/fashion/trend-radar` | Fetch current trends |
| `POST /api/fashion/occasion-planner` | Full occasion package generation |
| `POST /api/fashion/mood-style` | Mood → colors + outfits |
| `POST /api/fashion/try-on` | Virtual Try-On — takes person + outfit image, returns generated photo |
| `POST /api/food/fridge-scanner` | Detect ingredients + suggest recipes |
| `POST /api/food/meal-planner` | 7-day personalized meal plan |
| `POST /api/food/nuskha` | Traditional Indian remedies |
| `POST /api/mental/chat` | Persona + tone-aware chat with crisis detection. Body: `{ messages, language, persona, tone }` |
| `POST /api/mental/journal-insights` | Pattern analysis across mood journal entries |

## 🛡️ Safety & Ethics

- **Crisis keyword detection** (English + Hindi + Hinglish regex) instantly overrides the active persona/tone and surfaces Indian helplines: iCall `9152987821`, Vandrevala `1860-2662-345`, AASRA `9820466726`.
- **Medical remedies** ship with explicit disclaimers and "consult a doctor" warnings.
- **No PII stored** — journal lives in `localStorage`, chats are never persisted server-side.
- **Anonymous circles** — auto-generated handles, no accounts, no tracking.
